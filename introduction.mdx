## Overview

Air Tracker is a modern Angular-based frontend application that provides real-time aircraft tracking capabilities. Built with Angular 20 and powered by Leaflet maps, it delivers an intuitive interface for monitoring live flight data, exploring detailed aircraft information, and visualizing flight patterns on interactive maps.

## What's Inside

Air Tracker combines powerful features with a clean, responsive design:

- **Smart Polling System** — Adaptive polling intervals based on backend responses minimize server load while keeping data fresh
- **Responsive UI** — Desktop panels and mobile bottom sheets provide optimal viewing experiences across all devices
- **Rich Data Display** — Aircraft photos, flight paths, altitude, speed, and operator information at your fingertips
- **State Management** — Angular signals provide reactive state management for seamless real-time updates
- **Multiple Map Layers** — Switch between street, satellite, and terrain views

## Technology Stack

Air Tracker is built with modern web technologies:

- **Angular 20** — Latest Angular framework with standalone components and signals
- **Leaflet** — Interactive map library with custom marker animations
- **Angular Material** — Material Design components for UI consistency
- **RxJS** — Reactive programming for data streams and async operations
- **TypeScript** — Type-safe development with full IntelliSense support
